# Brent Oil Data Exploration

## Project Overview
This project focuses on the descriptive statistical analysis of Brent Oil prices over a five-year period. It demonstrates core data science skills including data loading, cleaning, filtering, and visualization.

## Objectives
* Load CSV data into a Pandas DataFrame.
* Perform data cleaning (handling missing values in the 'Vol' column).
* Generate descriptive statistics (Mean, Median, Mode, Standard Deviation).
* Filter data based on specific timeframes (e.g., December 2021).

## Technologies Used
* **Language:** Python
* **Environment:** Google Colab / Jupyter Notebook
* **Libraries:** Pandas, Matplotlib

## How to Run
1. Open the `.ipynb` file in Google Colab.
2. Mount your Google Drive to access the dataset.
3. Run the cells sequentially to see the analysis results.
